# Assignment 2c - Angular Register, Login and Profile

## Aim

Create an Angular application with:

- Register User
- Login User
- Show User Data on Profile Component

## Run

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Notes

This practical stores users in browser `localStorage`, so it can run without a backend. Register a user, login with the same email and password, then open the profile page.
