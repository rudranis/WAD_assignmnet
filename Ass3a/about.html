About

A small website served without Express.

This assignment shows how Node.js core modules can serve a complete static website.

Purpose

Why this server exists

The server listens for browser requests, maps each URL to a file inside this folder, reads that file, and sends it back with the correct content type.

Modules Used

  • http creates the web server.
  • fs reads files from disk.
  • path safely builds file paths.
  • URL reads the requested route.
